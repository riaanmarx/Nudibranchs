Naming conventions

When naming animals and plants, we use a binomial (two-name) naming system for genus (category), and species. Both of these must be in Latin, or something that sounds like Latin. Many names are derived from geographic locations, individuals' names and words from other languages.

In Latin, nouns also have gender and the genus- and species name of a species should be of the same gender. This makes for much confusion when a species from a male genus is moved to a female genus, at which point its species name must undergo a "gender-reassignment". Now some people are used to the old genus, old species combo, some use the new genus, but old species combo, and others still the new genus, new species combo...and a naming convension meant to simplify and standardize specie naming ends up creating more confusion than clarity.

Now, just to make things interesting - lets assume for a second that a species that need to be moved to a different genus (after dna studies for example) was the original defining species of its genus. Strictly speaking, this is not allowed, so ALL the other species in the genus need to be moved to a new/other genera. This happened with Cuthona/Trencesia sibogae and is still a source of heated debate amongst scientists today.

Genus and species names should be written in italic and although the genus is capitalized, the species name should not be.

When a specimin is identified but there is some doubt - say some feature normally associated with the species is absent - then we prefix the species name with "cf.". This is an abbreviation for the Latin word "confer" meaning "compare with", indicating that the specimin is comparable with the specified species.

If we are unable to identify a specimin, we record the species name as "sp.", but chances are pretty good there is going to be another unidentified specimin in the same genus...so we started using "sp.1", "sp.2" etc. Now of course if I call a specimin sp.1, there is no guarentee that someone else would have called it sp.1, so we added the source to the name: Aphelodoris sp.1 [Rudman] = Aphelodoris sp.4 [Johnson] (just an example).

As time goes by and more exhaustive research is done, some species will inevitably move from one genus to another. Some species may be proven to be the same - in which case the older species name is kept.

All this instability in naming species cause a lot of confusion and printed books don't exacly update the names once the ink hits the paper. This is where a live species catalog like this one starts to add value above a printed book: We are able to update the site with changes or additions and fix errors immediately and everyone have access to the updated information henceforth.

The genus and species classification is however only the lowest two orders of classification used in science. Lets have a quick look at the different hierarchy layers:
* Kingdom: plant, animal, fungi, bacteria & protists(single-celled organisms))
* Phylum: cnidaria(invertibrates), chordia(vertibrates), arthropods, molluscs etc. (40 different groups)
* Class: mammalia(mammals), actinopterygii(bony fish), chondrichthyes (cartilaginous fish) , aves (birds), amphibia (amphibians), reptilia (reptiles) etc.
* Order: carnivora(carnivors), primate(apes and monkeys), artiodactyla and rodentia(rodents) etc.
* Family: felidae (cats), canidae (dogs), ursidae (bears), and mustelidae (weasels) etc. * Genus
* Species
